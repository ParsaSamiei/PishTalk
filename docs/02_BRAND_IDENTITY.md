# About the Brand

Pishtalk is the official monthly meetup organized by Pishnam Robotics Institute.

Its identity combines technology, education, innovation and community.

The brand should feel modern and trustworthy without becoming overly corporate.

The website should communicate professionalism while remaining welcoming to students and experienced engineers alike.

Engineering atmosphere

---

# Design Philosophy

The interface should feel calm.

Whitespace is preferred over decoration.

Every visual element must have a purpose.

Animations should reinforce interactions rather than distract from them.

The interface should feel timeless rather than trendy.

# Accessibility

Minimum contrast ratio:

WCAG AA

Keyboard accessible

Focus indicators must always be visible.

Animations should respect prefers-reduced-motion.
