# Information Architecture

Version: 1.0

---

# Purpose

This document defines the complete structure of the Pishtalk website.

Every page, section, navigation item, user flow, and hierarchy is described here.

The goal is to ensure the website feels intuitive, minimal, and easy to navigate while remaining scalable for future growth.
